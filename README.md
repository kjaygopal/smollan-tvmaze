# 🎬 Smollan Movie Verse

A modern Flutter Movie & TV Show discovery application built using the TVMaze API.

This project was developed as part of a Flutter Internship Assignment for Smollan.

---

# ✨ Features

## Core Features
- Browse trending TV shows
- Dynamic show search with debounce
- Detailed show information page
- Hero animations between screens
- Add/remove favorites
- Offline favorite persistence using Hive
- Responsive UI for multiple screen sizes
- Smooth custom floating bottom navigation

---

# 🌙 Bonus Features
- Dark/Light mode using Provider
- Persistent theme storage with Hive
- Lottie animations for loading & empty states
- Cached network images
- Error handling & retry states

---

# 🛠️ Tech Stack

- Flutter
- Dart
- Provider
- Hive
- TVMaze REST API
- Cached Network Image
- Lottie Animations
